# Time Manager Bot
I wrote this bot for my personal usage! The goal of this bot is to simulate studying/working room with your friends, and have a focus time & rest time depend on your custom scheduling configuration.
You can also use this bot with [@my_study_timer_bot](https://t.me/my_study_timer_bot) id.
I also appreciate contributers to commit their ideas to this repository by opening a pull request and set me as assigne & reviewer of PR
